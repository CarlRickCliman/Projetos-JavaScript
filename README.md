# Projetos-JavaScript
Meu Repositório onde colocarei Projetos e Mini-Projetos de autoria própria durante meu aprendizado de JavaScript

## Projetos

### SISTEMA DE PONTOS DE APRENDIZADO
Calcula progresso de estudos com base em:

- Horas estudadas
- Dias estudados
- Dificuldade da matéria
- Revisão de conteúdo

### D20 DUEL
Simula uma batalha por turnos com base em:

- Rolagem de dado D20
- Sistema de vida dos jogadores
- Acertos e erros de ataque
- Dano crítico
- Determinação de vencedor após os rounds
- Duelo com 7 Rounds

### LISTA DE PRESENÇA
Sistema simples que simula uma lista de alunos com controle básico de presenças, com base em:

- Lista de alunos armazenada em array
- Percorrer a lista usando loop
- Condicional para associar cada aluno a um valor de presença
- Função para retornar a presença de cada aluno
- Exibição formatada no terminal com alinhamento em colunas

### RELATÓRIO DE ESTOQUE
Simula um sistema simples de gerenciamento de estoque para uma loja. O programa percorre uma lista de produtos, aplica descontos em categorias específicas, calcula informações do estoque e gera um relatório organizado no terminal.


- Cadastro de produtos utilizando Arrays de Objetos.
- Exibição de um relatório formatado dos produtos.
- Aplicação automática de desconto em uma categoria específica.
- Cálculo do valor total do estoque.
- Cálculo do valor médio dos produtos.
- Cálculo do valor individual do estoque de cada produto.
- Contagem de produtos cadastrados.
- Identificação de produtos com estoque zerado.
- Exibição de mensagens diferentes para singular e plural na quantidade de produtos sem estoque.
